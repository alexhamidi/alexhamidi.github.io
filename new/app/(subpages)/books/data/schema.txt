BOOK DATA STRUCTURE
===================

Source of Truth: app/(subpages)/books/data/[slug]/

Each book folder contains:
  - data.json (required): { title, author, date, rating, spineColor, textColor }
  - text.md (optional): markdown review content
  - cover.png (optional): book cover image
  - film.mp4 (optional): video content
  - images/ (optional): additional images

Static assets (cover.png, film.mp4, images/) are synced to public/books/[slug]/ via:
  ./scripts/sync-books.sh

Routes are generated dynamically:
  - /books → lists all books with carousel
  - /books/[slug] → individual book page
