Projects

  • TritonTool

    JS, React, Express, PostgreSQL

    TritonTool is a web application I built while navigating the UCSD enrollment process. I noticed there was no centralized platform to look up courses, so I developed an app that scrapes various UCSD department catalog sites to deliver course information to users with a simple search. The app also supports adding and organizing courses to a schedule, viewing and 'checking off' the prerequisites for your planned courses, and compiling resources found across the web.

  • PlayNext

    JS, Python, Pytorch, Redis

    Playnext is a command-line tool that allows you to train multiple neural network models with the low level audio characteristics of spotify tracks and get personalized recommendations. The models sample thousands of features from the track, including pitches, beats, and timbres. You can train models in bulk or by giving feedback on recommendations. Check out my Github profile to see both PlayNext and Spotibulk, another program I used to populate my DB with the ~300,000 pseudorandom songs used for sampling in this project.
